# calculadora-media
Calculadora para médias de 5 números.


# como funciona:
O sistema pede ao usuário 5 números inteiros utilizando WriteLine e os armazena em variáveis do tipo float através de ReadLines.

Ao receber os 5 números, o sistema calcula a soma das 5 variáveis e as divide pelo total de números digitados, nesse caso, por 5.
